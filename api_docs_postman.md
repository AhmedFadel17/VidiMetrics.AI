# 📖 VidiMetrics.AI API Documentation & Postman Collection

This document maps the core API endpoints of VidiMetrics.AI and provides a copy-pasteable Postman Collection v2.1.0 configuration for testing.

---

## 🗺️ API Endpoint Map

### 🔑 Authentication & Identity Server
Endpoints exposed by `VidiMetrics.IdentityServer` (default: `http://localhost:5145`):
- **OIDC Discovery**: `GET /.well-known/openid-configuration`
- **Authorize User**: `GET/POST /connect/authorize` (redirects to `/login` if unauthenticated)
- **Token Exchange**: `POST /connect/token` (exchanges authorization code or refresh token)
- **UserInfo**: `GET/POST /connect/userinfo` (requires active access token)
- **Logout**: `GET/POST /connect/logout`

### 📊 Business Logic API
Endpoints exposed by `VidiMetrics.API` (default: `https://localhost:7264` / `http://localhost:5000`):

#### 🎬 Story Engine Component
- **Shows**:
  - `GET /api/shows` (Get all shows with filters/pagination)
  - `GET /api/shows/{id}` (Get show by ID)
  - `POST /api/shows` (Create show)
  - `PUT /api/shows/{id}` (Update show)
  - `DELETE /api/shows/{id}` (Delete show)
  - `GET /api/shows/stats` (Retrieve aggregated Story Engine metrics)
- **Episodes**:
  - `GET /api/episodes` | `GET /api/episodes/{id}`
  - `POST /api/episodes` | `PUT /api/episodes/{id}` | `DELETE /api/episodes/{id}`
- **Characters**:
  - `GET /api/characters` | `GET /api/characters/{id}`
  - `POST /api/characters` | `PUT /api/characters/{id}` | `DELETE /api/characters/{id}`

#### 🤖 AI Assistance & Generation Component
- **Media Stats**:
  - `GET /api/ai/media/stats` (Retrieve user generation quotas, video stats, and limits)
- **Copilot**:
  - `GET /api/ai/copilot` (Trigger system chat copilots and assistants)
- **AI Assets**:
  - Scripts: `/api/ai/scripts` (Create and manage generated scripts)
  - Images: `/api/ai/images` (Create and manage generated images)
  - Videos: `/api/ai/videos` (Create and manage generated videos)

#### 📢 Channel Integrations Component
- **Channels**:
  - `GET /api/channels` (Get all channels)
  - `GET /api/channels/me` (Get logged-in user's channels)
  - `POST /api/channels` (Register/link channel)
  - `POST /api/channels/sync/{platform}/{channelId}` (Synchronize metrics from YouTube/TikTok)
